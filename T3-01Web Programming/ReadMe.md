# CSS
	There Was Tree Way To Write CSS:
	1-inline:
	<p style="color:red;">Hello</p>
	================
	2-Documant Level|
	================
	<html>
	<body>
	<style>
	p{
	color:red
	}
	</style>
	<p>Hello</p>
	</body>
	</html>
	================
	3-External Level|
	================
	<html>
	<link rel="stylesheet" type="text/css" href="mystyle.css">
	<body>
	<p>Hello</p>
	</body>
	</html>
	
#
# Type of CSS Selectors
	1-Element Selector:
	p{color:red;}
	
	2-Id Selector:
	#para1 {color:red;}
	
	3-Class Selector:
	.center {text-align:center;}
	
	4-Universal Selector:
	* {text-align: center;color: blue;}
	
	5-Pseudo Selector:
	a:link {  color: #FF0000;}
